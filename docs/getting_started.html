ARX Setup

Tested on Linux platform with Python 3.13.7

Start

Clone the repository with git clone https://github.com/vladimir-sama/arx-lang.git

Install gcc & llvm or use python arx.py install if you have chocolatey in Windows.

Run pip install -r requirements.txt or in a virtual environment if on Linux.

Usage

Run python arx.py build hello_world.arx where hello_world.arx is the Artemis code file you want to compile.
